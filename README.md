# Naruto Card Game — Exame Chunin
Repositório oficial do trabalho final da disciplina de Computação Gráfica (2026.1).

## Sobre o Projeto
Um jogo de cartas em turnos 3D inspirado no Hearthstone, ambientado na Arena do Exame Chunin do universo de Naruto. Os jogadores utilizam Chakra para invocar Ninjas e conjurar Jutsus. 

Projeto desenvolvido em Python utilizando **PyOpenGL (Moderno Core)**, **GLFW** para gerenciamento de janelas/inputs, e **GLM** para matemática de matrizes.
